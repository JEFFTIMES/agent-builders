# Apps v0.2 - agent-top3pm

## Core Systems in Confirmed Workflow
1. `Mantis` - TOP3 intake, dependency references, progress updates, deliverable publishing.
2. `Microsoft Teams` - two-tier group communication (Core / All) and meetings.
3. `Jenkins` - branch creation and image build execution.
4. `Outlook` - build success/failure notification channel.
5. `InfoSite` - built image/artifact retrieval.
6. `Phabricator` - QA progress/status tracking and validation context.

## Supporting System
1. `GitLab` - code integration/merge destination by Dev.
