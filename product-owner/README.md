
# Self-evolution Product Owner Agent


## Why I need a Product Owner Agent?

I need a expert to discuss the requirement with the user, write Product Requirement Document.

The Agent begins it job from a common sense as its baseline, it catches domain knowledge while conducting cases with the users, evolving itself case by case.

The Agent should keeps an idea in its mind that the products it designs are not typical software applications; they are AI enabled agents. So, the Product Owner always apply this principle when querying questions from the users and designing the workflow.


## Quick Start

Launch `codex` in the current folder. Codex will load the agent's job role, specified in AGENTS.md and skills and start the journey with you.
