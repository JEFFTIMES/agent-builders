# AI Agent Team Roles (Software Dev + Ops)

## Core Build Roles
1. Product Owner Agent: Turns goals into clear requirements, acceptance criteria, and scope boundaries.
2. Solution Architect Agent: Chooses system design, data flow, APIs, storage, and integration patterns.
3. Tech Lead Agent: Converts architecture into task breakdowns, coding standards, and risk callouts.
4. Feature Engineer Agents (23): Implement features and glue code; each owns a service or module.
5. QA/Test Agent: Writes test plans, generates test cases, and validates behavior.

## Ops & Reliability
6. DevOps/SRE Agent: Infrastructure-as-code, deployment pipelines, monitoring, incident playbooks.
7. Security/Compliance Agent: Threat modeling, secrets handling, access policies, audits.
8. Data/Telemetry Agent: Instrumentation, logs/metrics, dashboards, experiment tracking.

## Governance & Quality
9. Reviewer Agent: Code review, architecture review, regression risk assessment.
10. Documentation Agent: Maintains specs, runbooks, onboarding docs, and agent usage guides.

## Meta-Agent Roles (for creating other agents)
11. Agent Designer: Crafts agent prompts/specs, tools, and evaluation criteria for new agents.
12. Agent Evaluator: Benchmarks agent performance, creates test suites, and flags drift.
